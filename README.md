# 🏥 Hospital Emergency Room Dashboard (Excel + Power Query)

## 📌 Overview
This project analyzes hospital emergency room performance using structured data from **Hospital Emergency Room Data.csv**.  
The dashboard was created entirely in **Microsoft Excel**, leveraging **Power Query** for data cleaning and transformation.  
It provides monthly insights into patient demographics, admission trends, departmental referrals, and service efficiency.

---

## 📊 Key Metrics & Insights
- **Total Patients:** 4,878  
- **Average Wait Time:** 35.47 minutes  
- **Patient Satisfaction Score:** 4.96 / 5  
- **Admission Status:** 50% Admitted, 50% Not‑Admitted  
- **Gender Split:** 52% Male, 48% Female  
- **Attendance:** 60% Delayed, 40% On‑Time  
- **Age Group Analysis:** Highest patient volume in 20–29 and 50–59 age groups  
- **Department Referrals:** Majority (2,886) without referral; top departments include General Practice (966) and Orthopedics (532)

---

## 🛠 Tools & Techniques
- **Microsoft Excel**
  - Pivot Tables, Charts, and Slicers for interactive visualization  
  - Conditional Formatting for KPI highlights  
- **Power Query**
  - Data cleaning, merging, and transformation  
  - Automated refresh for monthly updates  
- **CSV Dataset**
  - Source data containing patient records and departmental details  

---

## 🎯 Business Impact
The dashboard helps hospital management:
- Identify delay patterns and optimize staffing schedules  
- Monitor patient satisfaction and wait‑time efficiency  
- Allocate resources effectively across departments  
- Improve overall patient throughput and service quality

## 📂 Repository Structure
- `Hospital_ER_Dashboard.xlsx` → Excel dashboard file

## 🚀 How to Use
1. Download the repository files.  
2. Open `Hospital_ER_Dashboard.xlsx` in **Microsoft Excel**.  
3. Use slicers to filter by **month, department, or age group**.  
4. Review KPIs and charts for actionable insights.
